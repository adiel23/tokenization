# Strategy Directory

This directory captures the project strategy at a glance:

- `mission-vision.md`
- `long-term-objectives.md`
- `action-plan.md`

Each document uses a simple wiki-style layout with:

- a short purpose statement
- a direct link to the corresponding wiki page
- a dated change log

## Update Notes

- Keep the content aligned with the active roadmap and issue backlog.
- Prefer concise, decision-oriented language over long narrative blocks.
- Record material changes in the document change log whenever strategy shifts.
